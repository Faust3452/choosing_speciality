# Описание работы программы
## Задача
```
Написать программу, которая из имеющегося массива строк формирует массив строк длина которых меньше либо равна 3 символа
```
## Алгоритм работы программы
```
Программа осуществляет:
1. ввод с клавиатуры массив строк;
2. затем в цикле осуществляется проход по всем элементам массива, вычисляется количество элементов массива, длина которых меньше либо равна 3 символам. 
3. после этого создается новый массив строк размером с вычисленным ранее количеством на шаге 2.
4. В цикле осуществляется проход по всем элементам исходного массива и те элементы, длина которых меньше либо равна 3, записываются в массив, созданный на шаге 3.
5. Вывод на экран нового массива с элементами, чья длина меньше либо равна 3.
```
## Описание программы, написанной на C#
```
В программе реализованы три функции:
1. void ArrPrint(string[] arr) - осуществляет вывод в консоль массива, который передается в качестве аргумента;

2. string[] ArrFill(int arrSize) - осуществляет заполнение массива значениями, введенными с клавиатуры. В качестве аргумента принимает требуемый размер массива и возвращает сформированный массив; 

3. string[] FindStrings(string[] arr) - осуществляет поиск строк с длиной меньше или равной 3 символам в массиве строк, который передается в качестве аргумента. Функция формирует массив из найденных строк и возвращает его.
```
